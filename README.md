# Expense Tracker 💰

A simple Expense Tracker web application built using HTML, CSS and JavaScript.

## Features

- Add income and expenses
- Delete transactions
- Calculate balance
- Track income and expenses
- Data saved using Local Storage
- Responsive design


## Technologies Used

- HTML5
- CSS3
- JavaScript


## How to Run

1. Clone repository

2. Open index.html

3. Run using Live Server
